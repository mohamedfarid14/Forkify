# Forkify
MVC Java Script Project provides the service of displaying recipes through forkify API and displays it with some features such as adding to favorites and displaying the recipe according to the number of individuals and the possibility of adding and reducing a component of the recipe

# Installing
```
To run the repo on your local machine 

clone the repo 
git clone https://github.com/mohamedfarid14/Forkify.git

Install dependencies
npm install 

then 
npm run start:dev 

click http://localhost:8080/ will open the project in browser  tab 
```
